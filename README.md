# Project Objective
A data analysis project exploring release patterns by year/quarter and genre, with an interactive Tableau dashboard and Excel cleaning steps.

# Dataset used
https://www.kaggle.com/datasets/jypenpen54534/steam-games-dataset-2021-2025-65k


## Tools Used
Excel (Power Query/ data cleaning)

Tableau (calculated fields, parameters, dashboard, filters)

### Questions(KPIs)
This analysis focuses on publishing activity, market structure, and pricing behaviour.

Volume KPIs
Total Releases
Releases by Year
Releases by Pandemic Phase

Market Structure KPIs
Genre Share
Publisher Contribution
Commercial KPIs
Average Game Price

Price Variation by Publisher / Genre

## Process
Excel (Cleaning & prep)
Removed duplicates and blanks
Standardised date formats (e.g., “Q4 2025” → real date)
Split/cleaned text fields (genres/tags)
